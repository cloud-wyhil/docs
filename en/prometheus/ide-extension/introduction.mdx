## IDE Extension

An extension system that can be added to an integrated development environment (VSCode, IntelliJ, etc.) for developers.

Prometheus supports using Prometheus in the IDE through the Extension.

By logging in, developers can configure their personal AI utilization environment, and settings and usage history are stored on the server.


### Main Features

- Chat
- Agent
- Rule
- Model
