# AI 코딩 온라인 학습 과정 소개

## 1. 과정 개요

본 과정은 인공지능(AI)을 활용한 코딩 능력을 체계적으로 학습할 수 있도록 설계된 실습 중심 온라인 교육 프로그램입니다.

AI 시대에는 더 이상 코드를 처음부터 모두 직접 작성하는 방식이 아니라,  
AI와 협업하여 더 빠르고 효율적으로 소프트웨어를 개발하는 능력이 중요합니다.

이 과정은 다음을 목표로 합니다.

- AI 기반 코딩 방식 이해
- 생성형 AI를 활용한 개발 생산성 향상
- 실무에서 바로 활용 가능한 AI 코딩 역량 습득
- 자동화 도구 개발 능력 확보

---

## 2. 과정 대상

이 과정은 다음과 같은 분들에게 적합합니다.

- 프로그래밍을 처음 배우는 입문자
- 기존 개발 방식에 AI를 도입하고 싶은 개발자
- 반복 업무를 자동화하고 싶은 실무자
- AI 코딩 도구를 업무에 활용하려는 기획자
- 업무 자동화 시스템을 직접 만들고 싶은 누구나

프로그래밍 경험이 많지 않아도 충분히 따라올 수 있도록 단계별로 구성되어 있습니다.

---

## 3. 학습 목표

과정을 수료하면 다음과 같은 역량을 갖추게 됩니다.

- AI를 활용한 코드 작성 능력
- AI 코딩 도구 활용 방법 이해
- Python 기반 자동화 프로그램 개발
- AI API 연동 프로그램 구현
- 실제 업무 문제 해결을 위한 AI 코딩 활용

궁극적으로는 “AI와 함께 일하는 개발자”로 성장하는 것이 목표입니다.

---

## 4. 과정 특징

### 4.1 실습 중심 교육

- 단순 이론이 아닌 실습 위주의 구성
- 실제 코드를 직접 작성하며 학습
- 업무 시나리오 기반 예제 제공
- 즉시 활용 가능한 샘플 코드 제공

### 4.2 단계별 학습 구조

- 기초 → 활용 → 응용 → 프로젝트 순서로 진행
- 누구나 이해할 수 있는 난이도 설계
- 반복 학습이 가능한 온라인 콘텐츠

### 4.3 최신 AI 도구 활용

- ChatGPT, Copilot 등 AI 코딩 도구 활용
- OpenAI API 실습
- 자동화 스크립트 작성
- AI 기반 애플리케이션 개발
