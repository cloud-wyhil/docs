# 빠르게 실습하기 준비

## 1. 시작하기

이제 AI는 일부 전문가만의 도구가 아니라, 모든 직무에서 활용해야 하는 필수 역량이 되었습니다.  
본 문서를 통해 AI를 이해하고, 직접 활용하고, 나만의 도구를 만드는 경험을 시작해 보시기 바랍니다.

---

> “AI를 아는 사람과 모르는 사람의 차이가 아니라,  
> AI를 활용하는 사람과 활용하지 못하는 사람의 차이가 경쟁력이 되는 시대입니다.”

---

## 2. 사전 준비 사항

실습을 위해 다음 환경을 권장합니다.

- 기본적인 컴퓨터 사용 능력
- Java / Python / NodeJS 개발 환경
- 인터넷 연결 환경
- OpenAI API 또는 유사 AI 서비스 계정

각 실습 단계에서 필요한 환경 구성 방법도 함께 안내합니다.
