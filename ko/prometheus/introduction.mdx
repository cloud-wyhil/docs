# Overview

개발자가 AI 보조 도구를 통해 높은 효율성을 경험하고 있으나, 클라우드 기반 서비스들은 핵심 자산인 코드 유출과 같은 보안 위협 및 시스템 파편화라는 한계를 지니고 있습니다.

PROMETHEUS는 이러한 문제를 해결하기 위해 온프레미스 구축 방식을 채택하여 완벽한 보안 환경을 제공하고, 파편화된 도구들을 하나로 통합해 개발자의 시간 낭비를 최소화합니다.

조직 내부 데이터를 학습하는 RAG 기반 AI 에이전트 기능을 통해 기업 맞춤형 안전한 개발 환경 지원을 제공합니다.

## 주요 특징

Prometheus(프로메테우스) 솔루션은 보안과 효율 최적화를 AI로 완성하는 차세대 개발 플랫폼으로, 주요 기능은 다음과 같습니다.

### 강력한 보안 및 Secure Code 강화 기능
* Full On-Premise 환경 기반: 중요 데이터를 다루는 시스템을 위해 기업 방화벽 내부에서 완벽하게 구동되는 환경을 제공하여 외부 네트워크 사용이 불가능한 곳에서도 안전한 개발이 가능합니다.
* 코드 보안 점검 및 분석: Java, Python, Ruby, C, C# 등 다양한 언어에 대해 상세한 보안 점검 규칙을 적용하고, 코드 저장소에 저장된 코드의 위반 사항을 점검합니다.
* AI 보안 코드 수정 지원: 보안 위반 사항에 대해 AI Assistant가 개선점을 확인하고 직접적인 수정 방안을 제시하여 보안 취약점을 효율적으로 해결할 수 있도록 돕습니다.
* 외부 AI 접근 차단: 보안을 위해 OpenAI나 Claude와 같은 외부 인공지능 서비스에 대한 접근을 차단하고 폐쇄망 내에서 통합된 AI 개발 환경을 구축합니다.

### 컨텍스트 기반 AI 지원 (AI Agent & RAG)
* 검색 증강 생성(RAG) 기술 활용: 모든 자료를 통합하고 문맥에 맞는 정보를 제공하기 위해 RAG Agent, Vector DB, Search Agent를 활용합니다.
* 지능형 AI Assistant: 단순한 코드 생성을 넘어 프로젝트의 모든 자료와 문맥을 이해하고 지원하는 AI 기능을 구축하여 작업의 모호성을 줄이고 재작업 시간을 단축합니다.

### 통합 개발 시스템 및 환경 관리
* 통합 환경 구성: 개발을 위한 다양한 구성 환경을 하나로 통합하여 제공하며, 이를 한곳에서 관리할 수 있는 시스템을 제공합니다.
* 대시보드를 통한 시각화: 개인 대시보드뿐만 아니라 플러그인, 저장소(Repository), LLM 모델 사용 현황 등을 시각적으로 모니터링할 수 있는 다양한 대시보드를 제공합니다.
* 관리 기능: 사용자 관리, 문서 관리, AI 사용 이력 및 채팅 내역 관리 등 개발 프로젝트 운영에 필요한 제반 기능을 통합적으로 관리할 수 있습니다.

### 자원 및 효율 최적화
* 개발 자원 최적화: 구축된 하드웨어 및 소프트웨어 자원의 사용 관리를 자유롭게 구성하고 운영할 수 있는 체계를 구축합니다.
* 생산성 향상: 여러 도구 간의 전환(Context Switching)으로 인한 생산성 저하 문제를 해결하고, 정보 탐색 및 새로운 기술 적용에 드는 시간을 단축합니다.
이 솔루션은 특히 보안이 최우선인 제조, 공공, 국방, 금융, 헬스케어 분야에서 안전하게 AI 개발 도구를 활용할 수 있도록 설계되었습니다
