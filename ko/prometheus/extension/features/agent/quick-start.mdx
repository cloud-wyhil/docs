# Quick Start

> Prometheus의 Agent Mode(에이전트 모드)를 사용하여 AI 기반 도구를 활용해 코드 변경, 버그 수정, 명령 실행 등을 자동으로 수행해 보세요. 이 도구는 자연어 명령에 따라 코드베이스를 수정할 수 있습니다.

Agent Mode(에이전트 모드)는 채팅 모델에 다양한 코딩 작업을 처리하는 데 필요한 도구를 제공하여 모델이 스스로 결정을 내리고 사용자가 수동으로 컨텍스트를 찾고 작업을 수행하는 수고를 덜어줍니다.

### How to Use Agent Mode

채팅 입력란 아래에 있는 모드 선택기에서 `Agent`으로 전환할 수 있습니다. 모드 선택기에는 세 가지 옵션이 있습니다:

* **Chat mode**: 아무런 도구도 사용하지 않고, 오직 대화만으로 진행했습니다.
* **Plan mode**: 안전한 탐색 및 계획수립을 위한 읽기 전용 도구
* **Agent mode**: 변경에 사용할 수 있는 모든 도구를 활용 합니다.

<Info>
  Agent Mode(에이전트 모드) 또는 Plan Mode(플랜 모드)가 `Not Supported`'지원되지 않음' 메시지와 함께 비활성화된 경우,
  선택한 모델 또는 공급자가 도구를 지원하지 않거나 Prometheus에서 아직 도구를 지원하지 않는 것입니다.
  자세한 내용은 [모델 블록](/customize/models)을 참조하십시오.
</Info>

<Tip>
  키보드 단축키 `Cmd/Ctrl + .`을 사용하여 모드를 빠르게 전환할 수 있습니다.
</Tip>

### How to Chat with Agent mode

Agent mode는 [Chat](/ide-extensions/chat/how-it-works) 모드와 동일한 인터페이스 내에서 작동하므로 동일한 [input](/ide-extensions/chat/quick-start#how-to-start-a-conversation)을 사용하여 메시지를 보내고 [`@` context providers](/ide-extensions/chat/quick-start#how-to-use--for-additional-context) 또는 [highlighted code from the editor](/ide-extensions/chat/quick-start#how-to-include-code-context)를 추가하는 것과 같은 컨텍스트를 제공하는 동일한 수동 방법을 계속 사용할 수 있습니다.


#### How to Use Natural Language with Agent mode

Agent mode(에이전트 모드)를 사용하면 자연어 명령을 제공하고 모델이 작업을 수행하도록 할 수 있습니다. 예를 들어 다음과 같이 말할 수 있습니다.

> Set the @typescript-eslint/naming-convention rule to "off" for all eslint configurations in this project

Agent mode(에이전트 모드)는 작업을 완료하는 데 사용할 도구를 결정합니다.

## How to Give Agent Mode Permission

기본적으로 Agent mode는 도구를 사용하려고 할 때 권한을 요청합니다. Agent mode가 도구 호출을 진행하도록 허용하려면 `Continue`를 클릭하고, 거부하려면 `취소`를 클릭하십시오.
