## IDE Extension

개발자를 위한 통합 개발환경 (VSCode, IntelliJ 등)에 추가하여 사용할 수 있는 확장 시스템

Prometheus는 Extension을 통하여 IDE에서 Prometheus를 사용할 수 있도록 지원합니다.

로그인을 통하여 개발자 개인별 AI 활용 환경을 구성하며, 설정 및 사용 이력은 서버에 저장됩니다.


### 주요 기능

- Chat 기능
- Agent 기능
- Rule 설정
- 모델 선택
