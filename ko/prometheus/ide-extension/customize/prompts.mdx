# 프롬프트 (Prompts)

> 프롬프트는 모델과 에이전트의 응답 방식을 결정하는 전문적인 지침입니다:

* 특정 작업이나 프레임워크에 대한 **상호작용 패턴을 정의**합니다.
* 특정 기술에 대한 **도메인 전문 지식을 인코딩**합니다.
* 조직의 관행에 맞춰 **일관된 가이드를 보장**합니다.
* 여러 에이전트 간에 **공유하고 재사용**할 수 있습니다.
* 팀 전체의 코드 일관성을 유지하는 **자동화된 코드 리뷰어 역할을 수행**합니다.
