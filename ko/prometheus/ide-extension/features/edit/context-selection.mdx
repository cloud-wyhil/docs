# 수정 모드에서의 컨텍스트 선택

> Prometheus의 수정 모드가 파일 내용과 언어 서버를 사용하여 관련 코드 컨텍스트를 선택하는 방법을 알아봅니다.

## 텍스트 입력 사용 방법

입력창에 질문이나 지침을 입력하는 것이 유일하게 필수적인 컨텍스트입니다.

## 수정 모드에 포함되는 컨텍스트

현재 파일의 **전체 내용**이 컨텍스트 파악을 위해 프롬프트에 포함됩니다. 모델은 강조 표시되거나 지정된 범위 내에서만 수정을 시도합니다.
