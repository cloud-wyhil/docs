# 수정(Edit) 기능 맞춤 설정 방법

> 워크플로우에 더 잘 맞도록 Prometheus의 수정(Edit) 기능을 맞춤 설정하는 방법을 알아봅니다.

## 활성 수정(Edit)/적용(Apply) 모델 설정 방법

수정(Edit) 및 적용(Apply) 요청에 사용할 특정 모델을 구성할 수 있습니다.

1. 메인 입력창 위의 점 3개 아이콘을 클릭합니다.
2. 큐브 아이콘을 클릭하여 "Models" 섹션을 확장합니다.
3. 드롭다운을 사용하여 수정(Edit) 및 적용(Apply)에 사용할 모델을 선택합니다.
