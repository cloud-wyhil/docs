# 채팅 모드 빠른 시작

> 코드 컨텍스트 공유, 코드베이스 검색, 생성된 솔루션을 파일에 적용하는 기능을 갖춘 Prometheus의 AI 채팅 어시스턴트를 IDE에서 바로 사용하여 코딩 문제를 해결해 보세요.

채팅을 활용하면 IDE를 벗어나지 않고도 AI의 도움을 쉽게 받을 수 있습니다. 설명을 듣고, 코드를 생성하며, 대화식으로 솔루션을 반복적으로 개선해 보세요.

## 채팅 사용 방법 - 기본 사용법

### 대화 시작하는 방법

채팅 입력란에 질문이나 요청 사항을 입력하고 Enter 키를 누르세요.

**예시:**

* "이 함수에 대해 설명해줘"
* "이 코드에서 에러 처리는 어떻게 해?"
* "이 컴포넌트에 대한 테스트 코드를 생성해줘"

### 코드 컨텍스트 포함하는 방법

에디터에서 코드를 선택한 후, 단축키를 사용하여 채팅에 포함시킬 수 있습니다.

* VS Code: `Cmd/Ctrl + L`을 눌러 선택한 코드를 채팅으로 전송
* JetBrains: `Cmd/Ctrl + J`를 눌러 선택한 코드를 채팅으로 전송

### @를 사용하여 추가 컨텍스트 추가하기

`@`를 입력하여 특정 컨텍스트를 포함할 수 있습니다:

* `@Files` - 특정 파일 참조
* `@Terminal` - 터미널 출력 포함

## 응답 활용하기

AI가 응답에서 코드를 제공하면 다음과 같은 액션 버튼이 표시됩니다:

* **Apply to current file (현재 파일에 적용)** - 선택한 코드를 대체합니다.
* **Insert at cursor (커서 위치에 삽입)** - 커서 위치에 코드를 추가합니다.
* **Copy (복사)** - 코드를 클립보드에 복사합니다.

## 채팅 활용 팁

### 새로 시작하기

빈 채팅창에서 `Cmd/Ctrl + L`(VS Code) 또는 `Cmd/Ctrl + J`(JetBrains)를 눌러 새로운 세션을 시작하세요.

### 상세하게 요청하기

다음에 대한 세부 정보를 포함하면 더 좋은 결과를 얻을 수 있습니다:

* 달성하려는 목표
* 제약 조건이나 요구 사항
* 선호하는 코딩 스타일 또는 패턴

### 반복 및 개선

첫 번째 응답이 완벽하지 않다면:

* 후속 질문을 하세요.
* 수정을 요청하세요.
* 추가 컨텍스트를 제공하세요.

## 채팅의 주요 사용 사례

### 코드 설명

이해가 되지 않는 코드를 선택하고 "이 코드는 무엇을 하나요?"라고 물어보세요.

### 버그 수정

에러 메시지를 포함하고 "이 에러를 어떻게 수정하나요?"라고 물어보세요.

### 코드 생성

원하는 내용을 설명하세요: "사용자 프로필을 표시하는 React 컴포넌트를 만들어줘"

### 리팩토링

코드를 선택하고 "이 코드를 어떻게 더 효율적으로 만들 수 있을까?"라고 물어보세요.
