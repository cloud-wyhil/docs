# 채팅 모드 컨텍스트 선택

> 텍스트 입력, 강조 표시된 코드, 활성 파일 등을 포함하여 Prometheus가 채팅 요청에 맞는 관련 컨텍스트를 선택하는 방법을 알아봅니다.

## 텍스트 입력 사용법

입력란에 질문이나 지침을 입력하는 것이 유일한 필수 컨텍스트입니다.

## 강조 표시된 코드 포함하는 방법

VS Code에서 `Cmd/Ctrl + L` 또는 JetBrains에서 `Cmd/Ctrl + J`를 눌러 선택한 강조 표시된 코드가 프롬프트에 포함됩니다.

## 활성 파일 포함하는 방법

요청을 보낼 때 `Opt/Alt + Enter`를 누르면 현재 열려 있는 파일을 컨텍스트로 포함할 수 있습니다.

## 특정 파일 포함하는 방법

현재 워크스페이스에서 `@Files`를 입력하고 파일을 선택하여 특정 파일을 포함할 수 있습니다.

## 코드베이스 검색

코드베이스 전체에 대한 이해를 높이려면 [에이전트 모드에 코드베이스 및 문서 인식시키기 가이드](/guides/codebase-documentation-awareness)를 참조하세요.

## 문서 사이트 포함하는 방법

문서에 대한 이해를 높이려면 [에이전트 모드에 코드베이스 및 문서 인식시키기 가이드](/guides/codebase-documentation-awareness)를 참조하세요.

## 터미널 내용 포함하는 방법

`@Terminal`을 입력하여 IDE 터미널의 내용을 컨텍스트로 포함할 수 있습니다.

## Git Diff 포함하는 방법

`@Git Diff`를 입력하여 현재 브랜치에서 변경한 모든 내용을 포함할 수 있습니다.

## 다른 컨텍스트 제공자 사용법

기본으로 제공되는 컨텍스트 제공자의 전체 목록은 [여기](/customize/deep-dives/custom-providers)에서 확인할 수 있습니다.
