# 채팅 맞춤 설정 방법

> 사용자의 워크플로우에 더 잘 맞도록 Prometheus의 채팅 기능을 맞춤 설정하는 방법을 알아봅니다.

## 채팅 맞춤 설정 방법

채팅을 맞춤 설정하는 방법에는 여러 가지가 있습니다:

* [규칙(Rules)](/customize/rules)을 추가하여 시스템 프롬프트를 통해 모델에 지속적인 지침을 제공합니다.
* 자주 반복하는 지침으로 워크플로우를 시작하려면 [프롬프트(Prompts)](/customize/prompts)를 만듭니다.
