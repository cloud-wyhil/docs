# 컨텍스트 선택

> 파일 내용, 언어 서버 정의, 임포트 및 최근 파일 기록을 사용하여 Prometheus의 에이전트 모드가 관련 코드 컨텍스트를 선택하는 방법에 대해 알아봅니다.

[채팅](/ide-extensions/chat/context-selection)과 동일한 방법을 사용하여 수동으로 컨텍스트를 추가할 수 있습니다.

도구 호출 응답은 자동으로 컨텍스트 항목으로 포함됩니다. 이를 통해 에이전트 모드는 이전 작업의 결과를 확인하고 다음에 무엇을 할지 결정할 수 있습니다.
