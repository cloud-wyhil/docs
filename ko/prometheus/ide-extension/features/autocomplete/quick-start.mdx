# Prometheus 자동 완성 빠른 시작

> 인라인 코드 제안 활성화 및 제안 수락, 거부 또는 부분 수락을 위한 키보드 단축키를 포함하여 IDE에서 Prometheus AI 자동 완성을 빠르게 시작하는 방법을 알아봅니다.

## Prometheus 자동 완성 활성화 및 사용 방법

자동 완성은 입력하는 동안 인라인 코드 제안을 제공합니다. 이를 활성화하려면 IDE 오른쪽 하단 상태 표시줄의 "Prometheus" 버튼을 클릭하거나 IDE 설정에서 "Enable Tab Autocomplete" 옵션이 선택되어 있는지 확인하십시오.

## 자동 완성을 위한 키보드 단축키

### 전체 제안 수락

`Tab`을 눌러 전체 제안을 수락합니다.

### 전체 제안 거부

`Esc`를 눌러 전체 제안을 거부합니다.

### 제안 부분 수락

더 세부적으로 제어하려면 `Cmd/Ctrl` + `→`를 사용하여 제안의 일부를 단어 단위로 수락합니다.

### 제안 강제 실행 (VS Code)

기다리지 않고 즉시 제안을 트리거하고 싶거나, 제안을 무시한 후 새 제안을 받고 싶은 경우 키보드 단축키 **`Cmd/Ctrl` + `Alt` + `Space`**를 사용하여 강제로 실행할 수 있습니다.
