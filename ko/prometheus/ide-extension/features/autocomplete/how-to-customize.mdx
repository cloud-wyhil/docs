# Prometheus 자동 완성 설정 맞춤화

> 사용자 설정, 구성 옵션 및 IDE의 AI 코드 제안을 개선하기 위한 조정을 포함하여 Prometheus의 자동 완성 동작을 맞춤 설정하는 방법을 알아봅니다.

Prometheus는 자동 완성 동작을 맞춤 설정할 수 있는 몇 가지 설정을 제공합니다. 사용자 설정 페이지(톱니바퀴 아이콘)를 방문하여 이러한 설정을 관리할 수 있습니다.

모든 구성 옵션과 그 영향에 대한 포괄적인 가이드는 [자동 완성 심층 학습](/customize/deep-dives/autocomplete)을 참조하세요.
