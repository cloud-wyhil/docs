## 이미지

### 마크다운 사용하기

[마크다운 구문](https://www.markdownguide.org/basic-syntax/#images)을 사용하면 아래와 같이 이미지를 추가할 수 있습니다.

```md
![제목](/path/image.jpg)
```

이미지 파일 크기는 5MB 미만이어야 합니다. 그 이상의 파일은 [Cloudinary](https://cloudinary.com/)나 [S3](https://aws.amazon.com/s3/)와 같은 서비스를 통해 호스팅하는 것을 권장합니다. 해당 URL을 사용하여 임베드할 수 있습니다.

### 임베드 사용하기

이미지에 대해 더 세밀한 맞춤 설정이 필요한 경우, [임베드](/writing-content/embed)를 사용하여 이미지를 추가할 수도 있습니다.

```html
<img height="200" src="/path/image.jpg" />
```

### iFrames

문서 내에 다른 HTML 페이지를 로드합니다. 주로 비디오를 임베드할 때 가장 많이 사용됩니다.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
