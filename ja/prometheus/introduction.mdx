## Welcome

開発者はAI支援ツールを通じて高い効率性を経験していますが、クラウドベースのサービスには、コア資産であるコード漏洩などのセキュリティ脅威やシステムの断片化という限界があります。

PROMETHEUSは、このような問題を解決するためにオンプレミス構築方式を採用し、完全なセキュリティ環境を提供し、断片化されたツールをひとつに統合して開発者の時間の浪費を最小限に抑えます。

組織の内部データを学習するRAGベースのAIエージェント機能により、企業がカスタマイズした安全な開発環境をサポートします。

## 認証 (Authentication)

すべてのAPIエンドポイントは、Bearerトークンを使用して認証され、仕様ファイルから取得されます。

```json
"security": [
  {
    "bearerAuth": []
  }
]
```
