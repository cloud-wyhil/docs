## インラインコード (Inline code)

`単語`や`フレーズ`をコードとして表すには、バッククォート (`) で囲みます。

```
`単語`や`フレーズ`をコードとして表すには、バッククォート (`) で囲みます。
```

## コードブロック (Code blocks)

3つのバッククォートでコードを囲むことで、[フェンス付きコードブロック](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks)を使用できます。先頭のバッククォートの後にプログラミング言語を指定すると、シンタックスハイライトが適用されます。オプションで、言語名の後にコードの名前を記述することもできます。

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
