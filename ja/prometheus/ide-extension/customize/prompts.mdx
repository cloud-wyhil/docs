# プロンプト (Prompts)

> プロンプトは、モデルやエージェントの回答方法を形作るための専門的な指示です。

* 特定のタスクやフレームワークに合わせた**インタラクションパターンを定義**します。
* 特定の技術に関する**ドメイン知識をコード化**します。
* 組織の慣行に合わせた**一貫したガイダンスを保証**します。
* 複数のエージェント間で**共有および再利用**が可能です。
* チーム全体で一貫性を維持するための**自動コードレビュアー**として機能します。
