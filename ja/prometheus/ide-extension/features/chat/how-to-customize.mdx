# チャットのカスタマイズ

> ワークフローに合わせて、Prometheusのチャット機能をカスタマイズする方法について学びます。

チャットをカスタマイズするには、主に以下の方法があります：

* [ルール](/customize/rules)を追加して、システムプロンプトを通じてモデルに永続的な指示を与える
* [プロンプト](/customize/prompts)を作成して、頻繁に繰り返す指示をワークフローとして開始できるようにする
