# オートコンプリートのクイックスタート

> IDEでPrometheusのAIオートコンプリートを素早く使い始める方法を学びます。インラインコード提案の有効化や、提案の承認、拒否、または部分的な承認のためのキーボードショートカットが含まれます。

## オートコンプリートの有効化と使用方法

オートコンプリートは、入力中にインラインコードの提案を提供します。有効にするには、IDEの右下にあるステータスバーの「Prometheus」ボタンをクリックするか、IDEの設定で「Enable Tab Autocomplete」オプションがチェックされていることを確認してください。

## オートコンプリートのキーボードショートカット

### 提案をすべて承認する

`Tab` キーを押すと、提案された内容をすべて承認します。

### 提案を拒否する

`Esc` キーを押すと、提案を拒否します。

### 提案を部分的に承認する

より細かく制御するには、`cmd/ctrl` + `→` を使用して、単語ごとに部分的に承認します。

### 提案を強制的に表示する (VS Code)

待たずにすぐに提案を表示させたい場合や、一度閉じた提案を再度表示させたい場合は、キーボードショートカット **`cmd/ctrl` + `alt` + `space`** を使用して強制的にトリガーできます。
