# オートコンプリート設定のカスタマイズ

> ユーザー設定、構成オプション、IDEでのAIコード提案を改善するための調整など、Prometheusのオートコンプリートの動作をカスタマイズする方法について学びます。

Prometheusは、オートコンプリートの動作をカスタマイズするためのいくつかの設定を提供しています。これらの設定を管理するには、ユーザー設定（歯車アイコン）ページにアクセスしてください。

すべての構成オプションとその影響についての包括的なガイドは、[オートコンプリートのディープダイブ](/customize/deep-dives/autocomplete)をご覧ください。
