    const regex = /\[([^\]]+)\]\(([^)]+)\)/g;
    const parts = [];
    let lastIndex = 0;
    let match;
    let key = 0;
    while ((match = regex.exec(text)) !== null) {
      if (match.index > lastIndex) {
        const beforeText = text.slice(lastIndex, match.index);
        if (beforeText) {
          parts.push(<span key={key++}>{beforeText}</span>);
        }
      }
      const [, linkText, url] = match;
      parts.push(<a key={key++} href={url} target="_blank" rel="noopener noreferrer" style={{
        color: "#0066cc",
        textDecoration: "underline"
      }}>
          {linkText}
        </a>);
      lastIndex = regex.lastIndex;
    }
    if (lastIndex < text.length) {
      const remainingText = text.slice(lastIndex);
      if (remainingText) {
        parts.push(<span key={key++}>{remainingText}</span>);
      }
    }
    return parts.length > 0 ? parts : text;

## ネクストエディット（Next Edit）モデル

次の編集箇所を予測する先回り型のコード予測機能として、Prometheusは専用の [ネクストエディット](/ide-extensions/autocomplete/next-edit) モデルをサポートしています：

**サポートされているネクストエディットモデル:**

* `mercury-coder-nextedit`: ネクストエディット予測に最適化されたプライマリモデル

ネクストエディットは、オートコンプリート用に互換性のあるモデルが設定されており、適切なアクセス権限がある場合に自動的に有効になります。
