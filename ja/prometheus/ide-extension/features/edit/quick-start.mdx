# エディット機能のクイックスタート

> Prometheusのエディット機能（Edit）を使い始めて、AIの提案を利用してファイル内のコードを直接、迅速かつ的確に変更しましょう。変更を承認または拒否するためのキーボードショートカットも備わっています。

## エディットの使い方

エディット（Edit）は、特定のコードやファイルを迅速に変更するための便利な方法です。コードを選択し、変更内容を説明すると、ファイル内にインラインで差分（diff）がストリーミングされ、それを承認または拒否することができます。

エディットは、以下のような小規模で的を絞った変更に推奨されます：

* コメントの記述
* ユニットテストの生成
* 関数やメソッドのリファクタリング

## エディットの起動方法

修正したいコードブロックをハイライトし、`Cmd+I` (Mac) または `Ctrl+I` (Windows/Linux) を押してエディットモードを起動します。コードをハイライトせずに `Cmd/Ctrl+I` を押すと、現在のカーソル位置にコードを挿入する動作になります。

エディットを起動したら、指示を入力する準備が整います。

### 指示（Instructions）の与え方

ハイライトしたコードに対してモデルに加えてほしい変更内容を説明します。エディットの場合、プロンプトは比較的短く簡潔にするのが良いでしょう。より長く複雑なタスクの場合は、[チャット](/ide-extensions/chat/quick-start)の使用をお勧めします。

### 変更の承認と拒否

提案された変更は、ハイライトされたテキスト内にインライン差分として表示されます。

各提案を順番に確認し、`Cmd+Opt+Y` (Mac) または `Ctrl+Alt+Y` (Windows/Linux) で承認、`Cmd+Opt+N` (Mac) または `Ctrl+Alt+N` (Windows/Linux) で拒否することができます。

また、`Cmd+Shift+Enter` (Mac) または `Ctrl+Shift+Enter` (Windows/Linux) で一括承認、`Cmd+Shift+Delete` (Mac) または `Ctrl+Shift+Backspace` (Windows/Linux) で一括拒否することも可能です。

同じハイライト箇所に対して新しい提案を依頼したい場合は、再度 `Cmd+I` (Mac) または `Ctrl+I` (Windows/Linux) を押して、モデルに再プロンプトを出すことができます。
