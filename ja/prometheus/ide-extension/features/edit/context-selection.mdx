# エディットモードのコンテキスト選択

> Prometheusのエディットモードが、ファイルの内容と言語サーバーを使用して、どのように関連するコードコンテキストを選択するかについて学びます。

## テキスト入力の使い方

入力ボックスへの質問や指示の入力は、唯一の必須コンテキストです。

## エディットモードに含まれるコンテキスト

コンテキストとして、現在のファイルの**全内容**がプロンプトに含まれます。モデルは、ハイライトされた範囲または指定された範囲のみを編集しようとします。
