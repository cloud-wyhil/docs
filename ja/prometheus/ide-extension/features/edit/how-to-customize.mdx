# エディット機能のカスタマイズ

> ワークフローに合わせて、Prometheusのエディット機能をカスタマイズする方法について学びます。

## エディット/適用モデルの設定方法

エディット（Edit）および適用（Apply）のリクエストに使用する特定のモデルを構成できます。

1. メイン入力欄の上にある「3つの点」をクリックします
2. 立体（キューブ）アイコンをクリックして「Models」セクションを展開します
3. ドロップダウンを使用して、エディットおよび適用に使用するモデルを選択します
