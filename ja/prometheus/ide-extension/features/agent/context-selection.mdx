# コンテキストの選択

> Prometheusのエージェントモードが、ファイルの内容、言語サーバーの定義、インポート、および最近のファイル履歴を使用して、どのように関連するコードコンテキストを選択するかについて学びます。

[チャット](/ide-extensions/chat/context-selection)と同様の方法を使用して、手動でコンテキストを追加することができます。

ツール呼び出しの回答は、コンテキストアイテムとして自動的に含まれます。これにより、エージェントモードは前のアクションの結果を確認し、次に何をすべきかを決定することができます。
