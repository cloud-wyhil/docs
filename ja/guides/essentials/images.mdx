## 画像

### Markdownを使用する

[Markdown構文](https://www.markdownguide.org/basic-syntax/#images) を使用すると、以下のコードで画像を追加できます。

```md
![タイトル](/path/image.jpg)
```

画像ファイルのサイズは 5MB 未満である必要があることに注意してください。それ以上の場合は、[Cloudinary](https://cloudinary.com/) や [S3](https://aws.amazon.com/s3/) などのサービスでホストすることをお勧めします。その後、その URL を使用して埋め込むことができます。

### 埋め込みを使用する

画像にさらに細かなカスタマイズを加えたい場合は、[埋め込み](/writing-content/embed) を使用して画像を追加することもできます。

```html
<img height="200" src="/path/image.jpg" />
```

<Tip>

Mintlify は [Markdown 内の HTML タグ](https://www.markdownguide.org/basic-syntax/#html) をサポートしています。これは、Markdown 構文よりも HTML タグを好む場合に役立ち、無限の柔軟性でドキュメントを作成することができます。

</Tip>

### iFrames

ドキュメント内に別の HTML ページを読み込みます。最も一般的には動画の埋め込みに使用されます。

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
